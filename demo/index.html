Welcome to diecutter's demo!

So you heard about diecutter? Let's give it a try!

  • HTML client for Sphinx documentation templates: fill in the form and generate Sphinx-based documentation layout.
  • But you can also use curl, wget or any HTTP client! Learn more about diecutter's API usage in the project's documentation.
  • Diecutter's demo API is available online at http://diecutter.io/api.
  • Templates of this demo are part of diecutter's source code.

About diecutter

Diecutter is an open-source project that allows you to deploy and use templates as a service.
