About diecutter

diecutter is a server that handles templates. When you post data to a template resource, you retrieve the generated content. Templates can be either single files (returns single files) or directories (returns archive). Learn more in the documentation.

diecutter is open-source template rendering software.

diecutter.io is the original online provider for diecutter.
